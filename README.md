## 概述

這是一個用於計算學期GPA與學期成績的網站。它可以根據用戶輸入的科目、成績和學分，計算出學期總學分、學期成績以及學期GPA，並且可算出這幾學期的總學分、平均成績以及平均GPA並顯示折線圖圖表。

**主要功能**

* 計算學期總學分
* 計算學期成績
* 計算學期GPA
* 提供高科與台灣的學期GPA計算方式
* 易懂的折線圖圖表
* 電子郵件驗證

**使用方式**

1. [訪問網站](http://203.64.95.42/C112151111/GPA_login.php)   
管理者帳號：admin
密碼：1111xixa1111
2. 輸入科目數量
3. 為每門科目輸入科目名稱、成績和學分
4. 提交後查看計算結果

**技術細節**

* 使用 PHP 編程語言開發
* 提供簡單的表單界面
* 自動計算每門科目的 GPA
* 根據高科標準分類
* 顯示學期總學分、學期成績、學期GPA 和高科學期GPA計算方式
* 使用電子郵件驗證方式
* 使用符號轉義防止SQ?注入
* 密碼使用哈希加密
